# Reservation

Welcome to Reservation.
This iOS app. is an example sources to develop iOS App. based on recent Swift 3 development environment.

## Machinsms

* Navigation & Pop-up
* Search
* Geocoding & Geolocation
* Text Input/Output
* Image fetch from Photo Library
* Data In/Out
* File Load/Save

## APIs

This example covers below APIs:

* UIKit
  * Controllers
    * UIViewController
      * UITableViewController
    * UISearchController
    * UINaviagationController
  * Views
    * TableView
      * TableViewCell
    * UIButton
    * UILabel
    * UITextField
    * UIImageView
  * Image
    * UIImagePNGRepresentation
  * String
    * NSAttributedString

* CoreData
  * NSManagedObjectContext
